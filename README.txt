
Step by step instructions:

    1. Download 'Eclipse IDE for Java Developers', version is Version: Mars.2 Release (4.5.2)
    2. Unzip the Eclipse zip file in a suitable folder on your "C" drive. Locate Eclipse.exe executable, double click the icon (file) to start the IDE
    3. Create workspace (File -> SwitchWorkspace -> Other), name your workspace suitably.
    4. Download the Selenium Server and Selenium Client Jar files into Referenced Libraries. 
    5. Start a new Java Project (File -> New -> Java Project) for JungleSocks test assignment.
    6. Based on QAEngineeringProble document requirement, I have created three test cases to cover it.
